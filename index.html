My interest in web development began back in the early 2000s when Neopets was still a very big deal. I learnt HTML and CSS so I could make colourful webpages for my virtual pets and show them off.

Today, I am completing a coding bootcamp for full-stack web development. Already I have built a number of responsive websites from the ground up, including a code quiz on JavaScript concepts, and a weather dashboard that can get you forecasts for any city in the world.

In April 2022, I took part in a fun little collaborative project. The app pairs a user with a Pokémon depending on that user's favourite music artist. Check it out below!
